# 210_Project

Compiling and running:

    Windows:

        javac -cp .\lib\jsfml.jar\;.\build -d .\src\build\ .\src\java\*.java
        java -cp .\lib\jsfml.jar;.\src\build; MenuMaker

    Linux(Havent tested but should work):

        javac -cp ./lib/jsfml.jar/:./build/ -d ./src/build/ ./src/java/*.java
        java -cp ./lib/jsfml.jar:./src/build: MenuMaker
