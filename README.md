# 210_Project

Compiling and running:

    Windows:

        javac -cp .\lib\jsfml.jar\;.\code\ClassFolder\ -d .\code\ClassFolder\ .\code\*.java
        java -cp .\lib\jsfml.jar;.\code\ClassFolder; MenuMaker

    Linux(Havent tested but should work):

        javac -cp ./lib/jsfml.jar/:./code\ClassFolder/ -d ./code/ClassFolder/ ./code/*.java
        java -cp ./lib/jsfml.jar:./code/ClassFolder: MenuMaker
