# 210_Project

Compiling and running:

    Windows:

        javac -cp .\lib\jsfml.jar\;.\build -d .\build\ .\src\java\*.java
        java -cp .\lib\jsfml.jar;.\build; MenuMaker

    Linux(Havent tested but should work):

        javac -cp ./lib/jsfml.jar/:./build/ -d ./build/ ./src/java/*.java
        java -cp ./lib/jsfml.jar:./build: MenuMaker
